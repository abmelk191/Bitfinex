## Bitfinex 介绍

Bitfinex是一家注册在英属维尔京群岛的加密货币交易所，主要服务专业交易者和机构投资者。平台支持超过200种加密货币的交易，包括比特币、以太坊等主流币种，以及各种山寨币和稳定币。

这个平台的特点是流动性比较好，订单深度也不错，适合大额交易。提供现货交易、保证金交易、衍生品交易等多种交易方式，还有借贷功能。界面相对专业一些，新手可能需要适应一下，但老手用起来会觉得功能齐全。

技术层面，Bitfinex用的是冷热钱包分离存储，大部分用户资金放在离线冷钱包里。虽然2016年被黑过一次，但后来加强了安全措施，并且全额赔偿了用户损失。

## Bitfinex 测评

从用户反馈和第三方评测来看，Bitfinex的评价比较两极化。

**优点方面：**
- 流动性确实不错，大额交易不容易滑点
- 支持的币种多，各种小众币也能找到
- 保证金交易和衍生品功能比较完善
- API接口稳定，适合量化交易
- 客服响应速度尚可，英文沟通比较顺畅

**争议方面：**
- 2016年的黑客事件让一些用户心有余悸，虽然后来赔偿了
- KYC验证流程比较严格，大额提现需要完整认证
- 界面对新手不太友好，功能太多容易迷路
- 曾经与Tether公司的关系引发过争议
- 部分地区用户无法访问，美国用户被限制

Reddit和Twitter上的讨论显示，专业交易者对Bitfinex的评价普遍较高，认为是"真正的交易者平台"。但普通用户可能会觉得门槛有点高，不如币安、欧易那些平台容易上手。

CoinMarketCap的评分是7.6/10，主要扣分在用户体验和历史安全事件上。但在流动性和交易深度方面，Bitfinex确实排在前列。

值得注意的是，Bitfinex在监管合规方面一直在改进，2021年与纽约总检察长办公室达成和解后，合规性有所提升。
